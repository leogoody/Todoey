# Todoey

Simple To-Do List app for iPhone to keep track of all your tasks.

This is a project from The App Brewery's Complete App Development Bootcamp.

### Technologies Used

- Storyboard UI
- MVC Design Pattern
- UIKit
- Cocoapods
- UITableView
- UserDefaults
- Core Data
- Realm
